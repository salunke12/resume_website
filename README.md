# Shreyas Salunke - Developer Portfolio 🚀

A modern, highly responsive, and aesthetically pleasing personal portfolio website showcasing my skills, experience, and projects. Designed with a dark theme and glassmorphism UI elements to provide a premium user experience.

🔗 **[View Live Demo Here](https://salunke12.github.io/resume_website/)**

## 🌟 Features

- **Modern Glassmorphism UI**: Beautiful semi-transparent frosted glass cards and dynamic gradient background glows.
- **Fully Responsive**: Architected with a mobile-first approach using the Bootstrap 5.3 grid layout system, ensuring perfect rendering on desktops, tablets, and phones.
- **Micro-Animations**: Smooth scroll reveal effects and a dynamic typing hero section to engage users interactively.
- **Direct Mail Integration**: Zero-backend `mailto:` integration to allow seamless networking through Gmail.

## 🛠️ Technology Stack

- **HTML5**: Semantic web structure.
- **CSS3 / Vanilla CSS**: All primary aesthetics, glass effects, animations, and custom typography (Google Fonts: Inter & Outfit).
- **Vanilla JavaScript**: DOM manipulations, scroll intercepts, and interaction logic (without heavy frameworks).
- **Bootstrap 5.3 (CDN)**: Utilized strictly for responsive layout scaffolding (Flexbox grid, sizing, utilities).
- **FontAwesome**: Scalable vector icons.

## 💻 Local Setup

Since this is a lightweight static site, no build steps or bundlers are required!

1. Clone the repository:
   ```bash
   git clone https://github.com/salunke12/resume_website.git
   ```
2. Navigate into the directory:
   ```bash
   cd resume_website
   ```
3. Simply double-click `index.html` to open it in your browser, or start a local server if you're using VSCode:
   - **VSCode Live Server Extension**: Right-click `index.html` > *Open with Live Server*
   - **Python Simple Server**: Run `python -m http.server 8000`
